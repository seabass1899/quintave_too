# Alignment Language + Streak Fix

## Changes

- Replaced user-facing daily “loop” language with “alignment” language where it referred to the required daily flow.
- Preserved “loop” only where it refers to unwanted behavior/thought patterns.
- Updated Today page labels:
  - Today’s Execution Loop → Today’s Alignment
  - Daily Loop Missed → Alignment Not Established
  - Reopen Today’s Loop → Resume Today’s Alignment
- Fixed Today-page streak display so a completed/locked day is reflected immediately as visible consistency.

## Streak model note

The Today page now counts any locked day as part of visible daily consistency. The stricter clean-streak rules used by the frequency/Gray Zone model remain separate.
